# **App Name**: SnackingTV

## Core Features:

- Video Playback: Play videos seamlessly within the app, supporting all major video formats.
- Vertical Swipe Navigation: Enable vertical swiping to navigate between videos in a feed.

## Style Guidelines:

- Primary color: Deep blue (#2962FF), for a modern and engaging look.
- Background color: Dark gray (#282828), providing contrast and focusing attention on the video content.
- Accent color: Violet (#7F56D9) used for interactive elements to maintain the balance between playfulness and readability.
- Headline font: 'Poppins' (sans-serif) for a contemporary, precise feel. Body font: 'Inter' (sans-serif) for readability.
- Use modern, minimalist icons for playback controls and other interactive elements.
- Full-screen video display with minimal UI elements overlaid for an immersive experience.
- Smooth transitions between videos when swiping.
